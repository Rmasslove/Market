#### 1.1 Ветви:

- `main` - главная ветвь
- `release/0.1.0` - ветвь с номером релиза, перед слиянием в main
- `dev` - ветвь разработки
- `feat-'name'` - ветвь для разработки фичи, name - имя фичи (пример: feat-filter)
